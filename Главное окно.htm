Данная програма предназначена для накопления и анализа данных о кредитных ставках разного типа.
Состав программы:
Изменение данных
Данная форма предназначена для фильтрации, изменения, удаления, добавления данных.
Также, в этой форме можно вывести отчет по разным видам ставок в файл для просмотра/печати.
Анализ данных
Данная форма предназначана для статистического анализа данных за выбранный период.
